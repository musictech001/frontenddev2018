this testing code demonstrates in three steps: how to transfer parameters between parent and child components using props.
last modified: 2018-07-17 by Maoshifu

to test, replace App.js and child.js by src in src-steps, and run npm start.

install npm modules:
$ npm install

alternatively, create a symbolic link to an existing npm modules folder:
ln -s ~/Developer/node_modules/ ./node_modules

start:
$ npm start
